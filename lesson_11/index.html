position: static

position: static – це обчислене значення яке призначається за замовчуванням. Воно має статичне позиціонування (тобто пасивне позиціонування), або іншими словами об'єкт буде строго зафіксований. Властивість position: static для об'єкту вказувати не треба, вона присвоюється для нього автоматично.

position: relative

position: relative – має відносне позиціонування. Об'єкт позиціонується відносно свого початкового місця положення (тобто відносно свого положення в коді). Його початкове місце розташування залишається за ним. Об'єкт ніяким чином не впливає на зовнішнє середовище.

position: absolute

position: absolute – це абсолютне (безумовне) позиціонування обє'кту. В цьому випадку обє'кт абсолютно позиціонований, при цьому інші обє'кти відображаються на веб-сторінці як абсолютно позиціонований обє'кт відсутній. Положення обє'кта задається за допомогою параметрів left, top, right, bottom. Також на положення впливає значення властивості position для батьківського елемента. Якщо для батьківського елемента властивость position прийме стан static або position не буде задано, то відлік координат буде вестися від краю вікна браузера. Якщо для батьківського елемента, для властивості position буде встановлено fixed, relative чи absolute, то відлік координат буде вестися від краю батьківського елемента.

position: fixed

position: fixed – це обчислене значення близьке до absolute, але на відміну від нього прив'язується до зазначених параметрів left, top, right, bottom на екрані та не змінює свого положення при прокручуванні веб-сторінки.

position: sticky

position: sticky – це гібридне поєднання обчислених значень static та fixed. Спочатку обє'кт веде себе як static але після того як скролл доходить до його кордону він стає fixed та прилипає до зазначеної позиції top або bottom.
